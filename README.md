# How to run the app
git clone https://github.com/SleepyReaperVK/MainBranch.git<br/>
cd MainBranch<br/>
npm install<br/>
npm start<br/>
**Make sure to have a running Android/iOS emulator or a phone with the [Expo app](https://expo.io/).**
